This planner gives you a quick first-pass layout for an aluminium extrusion simulator cockpit.

## How to use

1. **Setup** - Adjust the base, pedal tray, and steering column modules.
2. **2D side view** - Use the preview to check seat, wheel, and pedal relationships before building detailed mounts.
3. **Sharing** - The planner writes its current state into the URL so a setup can be bookmarked or shared.

OpenSimGear is still in early development, which means contributions can shape the project in visible ways. If you enjoy simulator
hardware, documentation, firmware, design systems, or tooling, there is room to help.

## Where you can contribute

- The project needs media resources logo, brand, social, etc..
- Design improvements in many areas
- Knowledge base, bring you knowledge to the people!
- We need proper legal setup, privacy policy, terms of use, etc.. or somebody that can review it.

## How to get started

If you already know where you can help, send a short message describing your idea, skill set, or the problem you want to
work on. Small improvements are valuable too: fixing confusing docs, suggesting better calculator UX, checking hardware
assumptions, or pointing out missing guides all help move the project forward.

If you have ideas please get in touch via email <a href={`mailto:${config.contacts.info}`}>{config.contacts.info}</a> or
join our Discord server through the <a href={config.social.discord}>OpenSimGear Discord invite</a>.

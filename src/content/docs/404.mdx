## Why this happens

A 404 page means the server is reachable, but this specific page does not exist at the address you opened. In practice,
that usually means one of three things: the URL was typed incorrectly, an old bookmark points to content that has been
renamed, or an external website linked to a page that no longer matches our current documentation structure. OpenSimGear
docs are still evolving, so some older links may no longer land on the exact page they originally referenced.

## Good places to recover

If you were looking for project basics, start with [Getting Started](/getting-started/). If you were trying to find a
specific guide, use the docs sidebar to browse the current sections for calculators, third-party builds, and core
documentation. If you arrived from a saved bookmark, try shortening the URL back to the nearest section and navigate
forward from there.

## Still cannot find it?

If the missing page seems like something that should exist, there is a good chance the content has moved, not vanished.
Check the relevant section in the navigation, review nearby pages, or visit the OpenSimGear GitHub organization from the
homepage for project repositories and related resources. When a broken link comes from our own site, reporting it helps
us repair the docs faster.

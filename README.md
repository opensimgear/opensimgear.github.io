# OpenSimGear Website

Documentation website for OpenSimGear, built with Astro and Starlight.

## Requirements

- Node.js
- pnpm

## Getting Started

```bash
pnpm install
pnpm dev
```

## Commands

- `pnpm dev` - start local dev server
- `pnpm build` - run Astro checks and production build
- `pnpm preview` - preview production build locally

## Tech Stack

- Astro 6
- Starlight
- Svelte 5
- Tailwind CSS v4

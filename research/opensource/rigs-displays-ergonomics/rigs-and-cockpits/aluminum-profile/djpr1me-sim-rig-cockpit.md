# DIY Aluminium Profile Sim Rig

- project: DIY Aluminium Profile Sim Rig
- maintainer_or_org: djpr1me
- component_category: rigs-and-cockpits
- subcategory: aluminum profile cockpit
- maturity_or_status: Personal build design; README includes cockpit updates through 2025, but project is still a shared

- key_features:
  - Compact 40 mm aluminum extrusion cockpit inspired by Sim-Lab GT1 Evo and P1-X geometry.
  - Integrated single-monitor stand, wheelbase deck, rotated pedal position, shifter, handbrake, and sequential shifter
    provisions.
  - Designed around commodity extrusion, T-nuts, angle brackets, connection plates, adjustable feet, and a separate
    automotive/racing seat.
- hardware_or_bom:
  - Repository includes budget PDF, dimensions PDF, May-CAD export, scene file, photos, and BOM-like tables in the
    README.
  - Uses 40x80 and 40x40 extrusion lengths, M8/M6 T-nuts, ISO7380 bolts, washers, angle brackets, plates, adjustable
    feet, and a BRIDE VIOS-style seat in the author's build.
- firmware_or_software_stack:
  - No firmware required.
  - May-CAD is used for the design file; sim hardware connects through each device's normal drivers.
- build_requirements:
  - Aluminum extrusion cutting/machining service or tools.
  - May-CAD on Windows if editing the original design file.
  - Hand tools for extrusion assembly and seat/control mounting.
- compatibility:
  - Author designed around Thrustmaster T500RS, T3PA-PRO, TH8A, TSS handbrake, DIY button box, and ultrawide monitor.
  - Later notes mention Moza R9 and MOZA HGP shifter fitting with small reinforcement updates.
- strengths:
  - Permissive public-domain dedication makes remixing simple.
  - Practical BOM and dimensions help builders order cut extrusion.
  - Integrated monitor stand reduces separate mount cost.
- limitations:
  - Single-builder validation, not a standardized kit.
  - 40x80 frame may need reinforcement for heavier direct-drive or high-force pedal loads.
  - May-CAD source workflow is Windows-centric.
- commercial_analogs:
  - Sim-Lab GT1 Evo
  - Sim-Lab P1-X
  - Trak Racer TR80
- fit_notes:
  - Good reference for a low-cost profile rig or for deriving extrusion lengths before using a local supplier.
  - Treat structural loads as builder responsibility, especially with high-torque DD wheels and load-cell/hydraulic
    pedals.

# Sim Racing Library For Arduino

- project: Sim Racing Library for Arduino
- maintainer_or_org: dmadison
- component_category: shifters
- subcategory: Arduino library and USB adapter shields for Logitech pedals/shifters
- maturity_or_status: Mature library with generated docs, examples, releases, and companion shield files.
- license: LGPL-3.0 for library; shield hardware license not verified [uncertain].
- repo_url: https://github.com/dmadison/Sim-Racing-Arduino
- docs_url: https://dmadison.github.io/Sim-Racing-Arduino/docs/
- source_urls:
  - https://github.com/dmadison/Sim-Racing-Arduino
  - https://dmadison.github.io/Sim-Racing-Arduino/docs/
  - https://dmadison.github.io/Sim-Racing-Arduino/docs/logitech_shifter.html
  - https://github.com/dmadison/Sim-Racing-Shields
- key_features:
  - Arduino library for interfacing sim racing equipment such as pedals, analog shifters, analog handbrakes, and
    Logitech shifters.
  - Supports Logitech Driving Force shifter, G27 shifter, G25 shifter, and Logitech pedal peripherals.
  - Open source shields connect Logitech shifters and three-pedal sets to SparkFun Pro Micro, with 3D printable case and
    custom USB identity.
- hardware_or_bom:
  - Arduino Leonardo/Pro Micro class board.
  - DE-9 connectors for Logitech peripherals.
  - Optional Sim Racing Shields PCBs and printed enclosures.
- firmware_or_software_stack:
  - Arduino library examples.
  - USB adapter firmware based on Arduino HID/joystick stack.
  - Doxygen docs and device-specific wiring pages.
- build_requirements:
  - Install library through Arduino IDE or zip release.
  - Wire supported device according to docs or build shield PCB.
  - Flash example adapter firmware and calibrate/test in OS.
- compatibility:
  - Logitech Driving Force, G25, G27 shifters.
  - Logitech two/three-pedal peripherals.
  - Generic analog shifters, pedals, and handbrakes.
- strengths:
  - Clean documentation and library API.
  - Best open adapter path for reusing Logitech shifters/pedals outside Logitech wheel bases.
  - LGPL license for software.
- limitations:
  - Adapter, not a new mechanical shifter.
  - Device-specific wiring means incompatible Logitech variants can be damaged if connected wrong.
  - Console use depends on game/adapter ecosystem [uncertain].
- commercial_analogs:
  - Leo Bodnar Logitech shifter/pedal adapters
  - Logitech G Racing Adapter
  - Ricmotech/Basherboards Logitech adapters
- fit_notes:
  - Catalog under shifters because the Logitech shifter adapter is the strongest use case; also relevant to pedals and
    handbrakes.

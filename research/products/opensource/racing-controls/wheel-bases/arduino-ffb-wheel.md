# Arduino-FFB-wheel

- project: Arduino-FFB-wheel
- maintainer_or_org: ranenbg, based on BRWheel by Fernando Igor and others
- component_category: wheel-bases
- subcategory: Arduino DirectInput FFB wheel firmware
- maturity_or_status: Active hobby firmware with releases; repository shows ongoing updates but no explicit license.
- license: No repository license found [uncertain].
- repo_url: https://github.com/ranenbg/Arduino-FFB-wheel

- source_urls:
  - https://github.com/ranenbg/Arduino-FFB-wheel
  - https://github.com/fernandoigor/BRWheel
- key_features:
  - Standalone DirectInput USB joystick with force feedback on ATmega32U4 boards.
  - Supports 4 analog axes plus steering encoder, 2 FFB axes, button matrix/shift registers, analog XY H-shifter, and
    HX711 load-cell support on Y axis.
  - 500 Hz update loop, EEPROM settings, configurable FFB output as PWM or DAC.
- hardware_or_bom:
  - Arduino Leonardo/Micro/Pro Micro.
  - Optical/magnetic encoders, analog pedals, optional shift registers, optional HX711 load cell, external ADC/DAC,
    motor driver suitable for selected PWM/DAC mode.
- firmware_or_software_stack:
  - Arduino/C firmware forked from BRWheel/AI-Wave lineage.
  - Includes old Arduino toolchain folders and helper programs.
- build_requirements:
  - Compile/flash firmware or use releases.
  - Configure firmware options for encoder, axes, buttons, shifter, FFB output, and calibration.
  - Reset old Windows HID calibration if upgrading across HID range changes.
- compatibility:
  - Windows DirectInput-style FFB joystick/wheel.
  - Supports analog H-shifter and multiple pedal inputs within same controller.
- strengths:
  - More complete FFB HID feature set than many small Arduino wheel sketches.
  - Supports pedals and H-shifter integration.
  - Useful reference for legacy DIY FFB USB descriptors/effects.
- limitations:
  - License absent, so reuse rights are unclear.
  - Arduino/ATmega32U4 limits torque-control fidelity and performance.
  - Docs assume deep firmware configuration.
- commercial_analogs:
  - Logitech G27/G29
  - Thrustmaster T150/T300
  - older consumer FFB wheels
- fit_notes:
  - Best treated as firmware reference or experimental low-cost FFB base, not as clean open-hardware product.

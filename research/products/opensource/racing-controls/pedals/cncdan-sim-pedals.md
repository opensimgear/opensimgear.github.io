# CNCDan Sim Pedals

- project: CNCDan Sim Pedals
- maintainer_or_org: dmcke5 / CNCDan
- component_category: pedals
- subcategory: laser-cut and 3D-printed load-cell/hall pedal set
- maturity_or_status: Active open-hardware pedal design; repo recently updated and moderately popular.
- license: CERN-OHL-S-2.0
- repo_url: https://github.com/dmcke5/SimPedals
- docs_url: https://github.com/dmcke5/SimPedals
- source_urls:
  - https://github.com/dmcke5/SimPedals
- key_features:
  - Interchangeable pedal bodies for any 1/2/3 pedal combination.
  - Hall-effect and load-cell sensor support across pedals.
  - Includes printed components, metal components, STEP, DXF, STL, and folding PDFs.
  - Optional standalone Arduino controller code and calibration.
- hardware_or_bom:
  - 625 bearings, magnets, hall sensors, 200 kg NA151 load cell, HX711 amplifier, M6 rod ends, retaining collars, 2020
    extrusion, brackets, M4/M5/M6 hardware.
  - Metal or printed pedal plates depending build route.
- firmware_or_software_stack:
  - Arduino 32u4 standalone code.
  - HX711 Arduino library.
  - Serial calibration stored after moving pedals through travel.
- build_requirements:
  - 3D printing and/or sheet-metal laser cutting/folding.
  - Arduino Leonardo/Pro Micro class board if used standalone.
  - Load-cell/hall sensor alignment and calibration.
- compatibility:
  - PC USB joystick if using included Arduino controller.
  - Could also be wired to other controller boards such as FreeJoy, Leo Bodnar, or wheel-base pedal inputs with suitable
    electronics [uncertain].
- strengths:
  - Proper open-hardware license.
  - Strong mechanical file set, including metal fabrication paths.
  - Flexible sensor and pedal-count choices.
- limitations:
  - Requires fabrication services or accurate metal work for strongest build.
  - Standalone software was noted by maintainer as less tested than the mechanical design.
  - No polished commercial tuning app.
- commercial_analogs:
  - Heusinkveld Sprint
  - VRS DirectForce Pro Pedals
  - Sim-Lab XP1
  - Simgrade VX-PRO
- fit_notes:
  - Best open pedal chassis found for builders comfortable sourcing metal cuts and sensors.

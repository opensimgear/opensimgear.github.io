# WirelessWheel

- project: WirelessWheel
- maintainer_or_org: MorGuux
- component_category: steering-wheels
- subcategory: NRF24 wireless wheel electronics
- maturity_or_status: Functional hobby project; updated but smaller and older than OpenSourceSimWheelESP32.
- license: GPL-3.0
- repo_url: https://github.com/MorGuux/WirelessWheel
- docs_url: https://github.com/MorGuux/WirelessWheel
- source_urls:
  - https://github.com/MorGuux/WirelessWheel
- key_features:
  - NRF24-powered wireless steering wheel with buttons, LEDs, Nextion display, and SimHub support.
  - Sender/receiver architecture using Arduino boards and RF modules.
  - Supports button matrix, rotary encoders, WS2812 LEDs, Nextion displays, and battery monitoring.
- hardware_or_bom:
  - ATmega32U4 HID Arduino for PC side.
  - NRF24L01 modules, receiver Arduino, button matrix, rotary encoders, WS2812 LEDs, Nextion display, battery monitoring
    hardware.
- firmware_or_software_stack:
  - Arduino firmware.
  - Uses FastLED, BatterySense, RF24, RF24Network, ArduinoJoystickLibrary, and Keypad.
  - SimHub support for LEDs/display integration.
- build_requirements:
  - Build sender and receiver electronics, pair RF modules, wire button/display/LED hardware, and package power cleanly
    in wheel.
  - Mechanical wheel/rim and paddle design not included.
- compatibility:
  - PC USB HID via ATmega32U4 receiver/sender stack [uncertain exact topology].
  - SimHub for telemetry-driven LEDs/displays.
- strengths:
  - Lightweight wireless pattern using common Arduino modules.
  - Good feature mix for older DIY GT/formula wheels.
  - GPL codebase with Arduino dependencies.
- limitations:
  - NRF24 reliability depends on RF environment and packaging.
  - Smaller ecosystem and less polished documentation than ESP32 alternative.
  - No integrated mechanical CAD.
- commercial_analogs:
  - Cube Controls wireless wheels
  - Simucube Wireless Wheel compatible rims
  - wireless button plates
- fit_notes:
  - Useful if builder prefers simple NRF24/Arduino parts over ESP32 BLE.
- last_checked: 2026-04-29

# OpenGC

- project: OpenGC
- maintainer_or_org: Damion Shelton / OpenGC project
- component_category: display-systems
- subcategory: glass cockpit instrument display software
- maturity_or_status: Dormant; home page last updated April 24, 2006.
- license: Example application GPL; libraries BSD-style per project home page
- repo_url: https://sourceforge.net/projects/opengc/
- docs_url: https://opengc.sourceforge.net/
- source_urls:
  - https://opengc.sourceforge.net/
  - https://sourceforge.net/projects/opengc/
- key_features:
  - Multi-platform, multi-simulator C++ toolkit for simulated flightdeck glass cockpit displays.
  - Provides libraries for data sources, navigation components, gauges, and an example FLTK application.
  - Designed to support different simulators and operating systems.
- hardware_or_bom:
  - PC plus one or more instrument displays/touchscreens.
  - No custom hardware BOM.
- firmware_or_software_stack:
  - C++ with FLTK example app.
  - Historical support for FSUIPC/WideFS and FDSConnect-style data sources.
- build_requirements:
  - Legacy C++ toolchain and dependencies; build process may need modernization.
  - SourceForge downloads because original CVS/SVN history was affected by server changes.
- compatibility:
  - Historical home cockpit/glass-cockpit use across flight simulators.
  - Modern MSFS/X-Plane/P3D compatibility would need validation or adaptation.
- strengths:
  - Early open-source basis for dedicated cockpit instrument displays.
  - Library-first approach can help custom gauge developers.
  - Separates instrument displays from the main simulator screen.
- limitations:
  - Very old and likely not maintained.
  - Modern simulator APIs, high-DPI displays, and OS builds likely require work.
  - Not a modern drag-and-drop instrument panel package.
- commercial_analogs:
  - Air Manager
  - RealSimGear avionics displays
  - Flight Illusion instrument systems
- fit_notes:
  - More useful as historical/reference code than a current cockpit display recommendation.
  - Worth noting because it maps to open display systems for glass cockpit panels.

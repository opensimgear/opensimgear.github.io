- project: OpenHarrier SJU-4A Ejection Seat

- component_category: seats-and-ergonomics

- key_features:
  - Replica Stencel SJU-4/A seat for AV-8B Harrier cockpit builds.
  - OpenHarrier page shows build photos and FreeCAD model.
  - Seat is modular relative to main console and side-console build.
- hardware_or_bom:
  - GitHub folder provides SJU-4A build instructions and design files.
  - Project approach uses wood, 3D prints, laser/CNC panels, and free/open CAD tools.
- firmware_or_software_stack:
  - Physical seat; Arduino electronics may be added for handles/switches if integrated into full cockpit.
  - FreeCAD is the key design/editing tool.
- build_requirements:
  - FreeCAD, woodworking/CNC or laser fabrication, 3D printing, fastening, finishing, and upholstery/cushion work.
  - Builder must verify personal posture and reach, since replica geometry can be less forgiving than automotive seats.
- compatibility:
  - Designed for OpenHarrier AV-8B cockpit.
  - Can inform other military cockpit seat builds if dimensions are adapted.
- strengths:
  - Specific open seat package for a less-common Harrier cockpit.
  - Build photos help bridge CAD to physical assembly.
  - Seat-first path lets builders start with ergonomics before full cockpit structure.
- limitations:
  - Aircraft-specific and niche.
  - No real safety function; simulator replica only.
  - Comfort, adjustability, and padding may need builder tuning for long sessions.
- commercial_analogs:
  - NeoEngress simulated ejection seats
  - Monstertech MPS-1 flight seat
  - Custom combat-flight cockpit seat fabrications
- fit_notes:
  - Good fit when AV-8B side-console and main-console dimensions matter.
  - Useful source for upright combat-flight posture; less useful for reclined GT/formula racing.

# Commercial Component Research

Commercial product notes for simulator hardware categories covered by `src/content/docs/docs/components`.

Each product file should use these fields where discoverable:

- `product`
- `manufacturer`
- `component_category`
- `subcategory`
- `status`
- `price_or_msrp`
- `region_or_availability`
- `official_url`
- `source_urls`
- `key_specs`
- `compatibility`
- `software_or_driver_ecosystem`
- `strengths`
- `limitations`
- `fit_notes`
- `similar_products`
- `last_checked`

Mark uncertain values with `[uncertain]`. Prefer official product pages, manufacturer docs, and official store listings.
